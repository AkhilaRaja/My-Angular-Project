 Booth Prabhari
Booth Prabhari is a mobile application to manage election at booth level in Kerala by manipulating voters list in a booth(Polling Station). The term "Booth Prabhari" means Polling Station In-charge. The users of this application are political parties in Kerala. This app assist political parties to analyse voters list in a scientific way with least error factors. It helps user to access a particular voter based on the district, local body, ward and polling station he/she belongs to. User can manage the adequate details of voter such as name, age, gender, address, phone number, voter Id, religion, caste, which political party he/she belongs to, whether he/she died or out of station which are highly usable for winning an election.

Main Features :

Search Voter
-> Helps to search voter in a booth using serial number or name or address of a particular voter.
-> User can edit and retrieve basic details of a voter.
-> User can call a particular voter if phone number is available.
-> User can put vote for a particular voter in this app if he appears physically for voting on election day which helps in predicting exit poll result.

Booth Status
-> Users can manage candidates in election.
-> Shows a vivid picture of current status of the booth like polling station details, total votes, death votes, out of station votes, expected poll, total polled etc.
-> User can also manage candidate details.
-> Gives a graphical representation of pre poll and exit poll result with leads count.

Filter Voter
-> Users can filter voters based on religion, caste, gender, candidate, dead, voted, out of station and age or any of the above.

By using this app political parties can manage the election in a systematic way and can predict their votes before and after election. So that they can work in such a way to tackle the cup.

Front end : Angular | Back end : SpringBoot | Database : Firebase
