The users of this application are political parties in Kerala. This app assist political parties to analyse voters list in a scientific way with least error factors. It helps user to access a particular voter based on the district, local body, ward and polling station he/she belongs to. User can manage the adequate details of voter such as name, age, gender, address, phone number, voter Id, religion, caste, which political party he/she belongs to, whether he/she died or out of station which are highly usable for winning an election.
This application helps admin to manage district, local body, ward, polling station, religion, caste, candidate, voters list etc with a promising UI made up of angular material.
